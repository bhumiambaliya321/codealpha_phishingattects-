# 🎣 Phishing Awareness Training

> **CodeAlpha Cyber Security Internship - Task 2**

## 📖 Project Overview

This project was developed as part of the **CodeAlpha Cyber Security Internship**.

The objective of this project is to educate users about phishing attacks, social engineering techniques, fake websites, malicious emails, and cybersecurity best practices. The presentation aims to increase awareness and help users identify and prevent phishing attempts in real-world situations.

---

## 🎯 Objectives

* Understand phishing attacks
* Learn different types of phishing
* Recognize phishing emails and fake websites
* Understand social engineering techniques
* Learn cybersecurity best practices
* Improve online safety awareness

---

## 📚 Topics Covered

* Introduction to Phishing
* Why Attackers Use Phishing
* Types of Phishing Attacks
* How Phishing Works
* Signs of a Phishing Email
* Fake Website vs Real Website
* Social Engineering
* Real-World Phishing Examples
* Cybersecurity Best Practices
* How to Respond to Phishing Attacks
* Interactive Awareness Quiz
* Conclusion

## 🛡 Cybersecurity Best Practices

* Verify the sender's email address
* Never click suspicious links
* Check website URLs carefully
* Enable Two-Factor Authentication (2FA)
* Use strong and unique passwords
* Keep software updated
* Never share OTPs or passwords
* Report suspicious emails immediately
